Array.reverse()

							
							let listOfNumbers = [ 1, 2, 3, 4, 5 ];
							
						
							
							listOfNumbers.reverse();
							
						
							
							console.log( listOfNumbers );

Will output:

							
							[ 5, 4, 3, 2, 1]

WARNING: reverse() will mutate (change) the contents of the Array!
